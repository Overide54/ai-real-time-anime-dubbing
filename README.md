# AI Real-Time Anime Dubbing System

An AI-powered application that converts Japanese speech into English text and speech.

## Features

- Japanese Speech Recognition using Whisper
- Japanese → English Translation
- English Voice Generation
- Streamlit Web Interface

## Tech Stack

- Python
- Streamlit
- OpenAI Whisper
- MarianMT
- gTTS
- Transformers

## Future Roadmap

- Real-time microphone translation
- Video dubbing support
- Voice cloning
- Lip synchronization
- Multi-language output (English, Hindi, Telugu)
